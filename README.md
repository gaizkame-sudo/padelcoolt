# SE QLD Dashboard

Desktop dashboard for tides, surf, email, and AI/LLM news.

## Run

1. `npm install`
2. `npm start`

## Setup

- Surf providers: Stormglass or Swellcloud (choose in Settings).
- Stormglass API key: Add in Settings to enable Stormglass surf reports.
- Swellcloud API key: Add in Settings to enable Swellcloud surf reports.
- Gmail OAuth: Create OAuth client credentials and add Client ID/Secret in Settings.
- Outlook OAuth: Register an app in Azure and add Client ID and Tenant ID.

Email OAuth flows are scaffolded and will need implementation to pull inbox data.
